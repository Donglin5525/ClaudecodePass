# Changelog

本仓库重要变更记录。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/)。

## 2026-07-07

### Fixed
- **堵死授权弹窗超时偷偷放行（致命修复）**：macOS 弹窗超时或异常关闭时，原先静默回退，可能在用户没看到弹窗时被默认放行。现改为响铃提醒 + 切回终端 + 回退终端原生权限弹窗持续等待用户 —— 绝不替用户同意，也不直接拒绝（避免操作失败要重发）。
- **macOS 弹窗默认按钮由「授权」改为「拒绝」**：防止误触回车、窗口切换时的焦点事件等导致操作被意外放行。

### Changed
- **补全只读管道组合自动放行**：`find ... | xargs grep` 这类常见只读组合此前被误判为不安全而反复弹窗，现自动放行；`xargs rm` 等危险组合因 `rm` 不在只读清单而仍不放行。

## 2026-06-29

### Added
- **跨平台支持**：新增 Windows（ctypes MessageBox）弹窗、提示音、终端激活实现，与 macOS 并列。
- **首版发布**：Claude Code Permission Hook —— 自动放行只读/已知安全命令、MCP 工具，仅对计划文件编辑、未知/危险命令弹窗确认；管道引用感知、cd 前缀剥离。
